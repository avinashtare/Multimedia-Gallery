## Tech Stack

- **React** - Frontend framework
- **Redux** - State management
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **Unsplash API** - Photo source
- **Tenor API** - GIF source
- **Pexels API** - Video source

## Components

### Core Components

- `SearchBar` - Search input with modern design
- `Tabs` - Navigation between Photos, GIFs, and Videos
- `ImageGrid` - Displays photo results
- `GifGrid` - Displays GIF results
- `VideoGrid` - Displays video results
- `LoadingScreen` - Loading state with spinner or skeleton

### Card Components

- `ImageCard` - Individual photo card with hover effects
- `GifCard` - GIF card with play/pause toggle
- `VideoCard` - Video card with playback controls

## Installation

```bash
# Clone the repository

# Install dependencies

# Start the development server
npm start
```
